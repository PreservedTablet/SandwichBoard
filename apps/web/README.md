# @sandwichboard/web

SvelteKit 2 / Svelte 5 dashboard. Phase 0 ships a skeleton; the library,
leaderboard, inbox, and approval screens land in Phases 1–5
(see `docs/plan/06-BUILD-PLAN.md`).

Run from the repo root: `infisical run --env=dev -- pnpm dev`
(or `pnpm dev:local` with a gitignored `.env`).
